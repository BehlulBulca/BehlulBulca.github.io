## Overview

This is an **AI-powered financial analytics platform** designed for **Borsa Istanbul (BIST)** markets. It provides institutional-grade analytics, risk metrics, and machine learning-driven price forecasting for Turkish capital markets.

## Tech Stack


 - **Backend** - Python 3.12, Flask 3.0, Peewee ORM, APScheduler 
 - **Frontend** - Next.js 16 
 - **ML/AI** - TensorFlow LSTM models, Statsmodels econometrics, Ollama LLM via MCP 
 - **Database** - PostgreSQL 
 - **Visualization** - Lightweight Charts, Tailwind CSS, shadcn/ui

## ML Pipeline

The platform includes a complete ML pipeline with:

1. **Data Processing** â€” Ingestion, validation, feature engineering with deterministic seeding
2. **Model Management** â€” Training orchestration, version registry, rollback support
3. **Inference Engine** â€” Batch and single-symbol predictions with caching
4. **Database Sync** â€” Persistence for predictions, metrics, and job logs
5. **Monitoring** â€” Health dashboards and alert routing

## Background Jobs

- `DailyPredictionJob` | Daily | Refresh features and run inference |
- `WeeklyTrainingJob` | Weekly | Retrain LSTM models and version them |
- `DataIntegrityJob` | Scheduled | Validate data completeness and anomalies |
- `PerformanceReportJob` | Periodic | Aggregate KPIs and push notifications |
